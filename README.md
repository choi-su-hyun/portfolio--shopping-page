# Chair shopping site(portfolio)
의자를 판매하는 쇼핑몰 사이트 개발 
데이터베이스 설계, 프론트, 백 개발을 통해 React와 Express를 경험했습니다. React 위주로 처음으로 작업해보았던 포트폴리오 입니다.

**Back Github**
